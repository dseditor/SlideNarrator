# 簡報旁白講稿生成提示詞

請根據以下簡報內容，為每一頁生成口語化的繁體中文旁白講稿。

## 格式要求（非常重要，請嚴格遵守）

1. 每頁以 `Page數字:` 開頭（例如 `Page1:`、`Page2:`）
2. 每頁的所有旁白句子寫在同一行，用**空格**分隔
3. `Page數字:` 後面緊接內容，不要換行
4. **全部使用繁體中文**，不要包含英文或數字（如需提及英文術語，請用中文音譯或意譯）
5. 每頁 3-6 個句子為佳
6. 句末不需要加標點符號
7. 每個句子控制在 15-30 個字以內

## 正確範例

```
Page1: 各位同學大家好 今天我們要來討論的主題是人工智慧的發展 這是一個非常重要的議題

Page2: 首先讓我們來看一下背景資料 在過去十年間人工智慧技術有了飛躍性的進步 從語音辨識到影像處理都取得了突破性的成果

Page3: 在這張圖表中我們可以看到 市場規模從兩千年的一百億成長到了目前的一兆 這個成長趨勢預計還會持續下去
```

## 注意事項

- 語氣要自然，像是在現場對觀眾說話
- 數字請用中文寫出（例如：「兩千零二十四年」而非「2024年」）
- 專有名詞請用中文翻譯或音譯
- 如果簡報中有圖表，請加入「在這張圖表中」等引導語
- 每頁的句子數量與簡報該頁的內容量成正比
